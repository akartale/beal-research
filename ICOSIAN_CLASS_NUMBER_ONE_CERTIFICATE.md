# Exact class-number-one certificate for the maximal icosian order

## Setup

Let

\[
F=\mathbf Q(\sqrt5),\qquad \mathcal O_F=\mathbf Z[a],\qquad a^2-a-1=0,
\]

and let

\[
B=(-1,-1)_F.
\]

By `ICOSIAN_ORDER_CERTIFICATE.md`, the implemented icosian order \(R\subset B\) is maximal at every finite prime and \(B\) is ramified exactly at the two real places.

The exact arithmetic check is implemented in

```text
scripts/icosian_class_number_certificate.py
```

## The field class number

The discriminant of \(F\) is `5`. For a real quadratic field, Minkowski's bound is

\[
\frac{2!}{2^2}\sqrt{|D_F|}=\frac{\sqrt5}{2}<2.
\]

Every ideal class therefore contains an integral ideal of norm at most \(\sqrt5/2\). Since ideal norms are positive integers, that norm must be `1`, so every ideal class is principal. Hence

\[
h_F=1.
\]

## The special zeta value

For the quadratic character \(\chi_5\) modulo `5`,

\[
\zeta_F(s)=\zeta(s)L(s,\chi_5).
\]

The exact generalized Bernoulli computation gives

\[
B_{2,\chi_5}=\frac45,
\qquad
L(-1,\chi_5)=-\frac25.
\]

Since \(\zeta(-1)=-1/12\),

\[
\zeta_F(-1)=\frac1{30}.
\]

The script prints

```text
B2_chi5 4/5
L_minus_1_chi5 -2/5
zeta_F_minus_1 1/30
```

## Eichler mass

For a maximal order in a totally definite quaternion algebra over a totally real field of degree `2`, with no finite ramified primes, the Eichler mass is

\[
\operatorname{Mass}(R)
=2^{1-[F:\mathbf Q]}h_F\lvert\zeta_F(-1)\rvert.
\]

Therefore

\[
\operatorname{Mass}(R)
=\frac12\cdot1\cdot\frac1{30}
=\frac1{60}.
\]

## Weight of the implemented ideal class

The exact norm-one unit enumeration used by the repository gives

\[
|R^1|=120.
\]

The central norm-one units are exactly

\[
\mathcal O_F^\times\cap R^1=\{\pm1\}.
\]

Moreover, every totally positive unit of \(\mathcal O_F\) is a square: with fundamental unit \(a\), the totally positive unit group is generated by \(a^2\). Thus scaling by a central unit identifies

\[
R^\times/\mathcal O_F^\times
\simeq
R^1/\{\pm1\}.
\]

Consequently,

\[
\left|R^\times/\mathcal O_F^\times\right|=60,
\]

and the mass contribution of the principal right-ideal class is

\[
\frac1{60}.
\]

This equals the full Eichler mass. Since every right-ideal class contributes a positive reciprocal stabilizer weight, there can be no additional class.

Therefore

\[
h(R)=1.
\]

Equivalently, every locally principal right \(R\)-ideal class is represented by \(R\) itself.

## Consequence for the Brandt quotient

Because the maximal order has one right-ideal class, the global definite quaternionic double-coset set at standard level \(K_0(\mathfrak N)\) is represented by the single maximal-order class. Together with `OPEN_COMPACT_K0_CERTIFICATE.md`, this removes the missing ideal-class summation from the repository's projective-line model.

The remaining global issue is the exact passage from the norm-one quotient to the prescribed trivial-central-character automorphic quotient and then the Hecke-equivariant Jacquet--Langlands/newspace comparison. This certificate does not by itself settle oldforms, Eisenstein constituents, multiplicities, or residue-field extension.
