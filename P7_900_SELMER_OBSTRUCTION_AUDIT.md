# Selmer audit for the final quadratic survivor `(90,0)`

## Setup

After twisting by the quadratic ray character `(90,0)`, a reducible residual representation has semisimplification

\[
1\oplus\bar\varepsilon_7.
\]

The two nonsplit orientations are governed by classes in

\[
H^1(F,\mathbf F_7(1))
\quad\text{and}\quad
H^1(F,\mathbf F_7(-1)),
\]

with the actual finite-flat condition at the unique prime above `7` and the conductor conditions at the primes above `3` and `5`.

## Global arithmetic data

For

\[
F=\mathbf Q(\sqrt5),
\qquad
M=F(\zeta_7),
\]

the exact PARI/GP script

```text
scripts/p7_selmer_global_data.gp
```

computes

```text
[M:Q] = 12,
Cl(M) = 1,
rank O_M^x = 5.
```

Thus there is no `7`-primary class-group contribution in `M`. This is useful for the opposite extension orientation, but it does not by itself determine the ray/local eigenspaces.

## The `F_7(1)` direction is nonzero

The class number of `F` is one and its unit group modulo torsion is generated by

\[
w=\frac{1+\sqrt5}{2}.
\]

The exact script

```text
scripts/p7_fundamental_unit_local_class.py
```

works in

\[
(\mathbf Z/n\mathbf Z)[w]/(w^2-w-1)
\]

and finds seventh roots of `w` modulo the relevant prime powers:

```text
mod 3  : one root
mod 9  : one root
mod 5  : one root
mod 25 : one root
mod 49 : no root
```

Explicitly, roots are

```text
mod 9  : 8+4w,
mod 25 : 21+12w.
```

Hence the fundamental unit class is locally trivial at the primes above `3` and `5`, but nontrivial at the inert prime above `7`.

Because it is represented by a unit at `7`, it lies in the natural finite-flat Kummer local condition for extensions

\[
0\to\mathbf F_7(1)\to V\to\mathbf F_7\to0.
\]

Therefore the natural Selmer group in this orientation is not zero: it contains the class of `w`.

## Consequence

A strategy asserting that both extension Selmer groups vanish cannot close Lemma D.

The remaining reducible possibilities include at least:

1. the split representation `1 direct_sum epsilon_bar_7`;
2. the nonsplit finite-flat extension represented globally by the fundamental unit class;
3. possibly the opposite extension orientation, pending an exact ray-eigenspace computation.

To finish noncircularly, one must now do one of the following:

- compute the finite-flat Honda/Serre--Tate invariant of the Frey representation at `7` and show that it is neither the split class nor the local image of the fundamental unit class;
- compute the opposite Selmer orientation exactly and then combine it with the same local invariant;
- prove an integral Hilbert Eisenstein theorem applicable to this precise square-level stabilization;
- prove a reducible-compatible exact-level modularity theorem, allowing the existing level-`(2,2)` Brandt certificate to be used.

## Status

This audit narrows the Selmer route but does not close Lemma D. It proves that the most naive Selmer-vanishing approach is impossible and identifies the explicit surviving global extension class that any local argument must eliminate.
