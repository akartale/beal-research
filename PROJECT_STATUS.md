## The last reducible case

After the character and local checks, one case is left:

```text
(90,0)
```

The level-`(2,2)` Brandt calculation finds no matching mod-`7` eigensystem. It is tempting to use that result right away, but the level-lowering theorem we would cite already assumes irreducibility. So that shortcut is not valid.

There are two possible noncircular routes:

1. **Recover the explicit RM correspondence.** This is blocked for now because the published quotient does not give the RM action or the missing matrix.
2. **Finish the weight-two Eisenstein argument.** This looks like the better route. The expected constant-term factor is

   ```text
   zeta_F(-1)(1-N(3))(1-N(sqrt(5))) = 16/15,
   ```

   which is a `7`-adic unit.

Five concrete checks are still missing:

- build the right stabilization at `(3)^2(sqrt(5))^2`;
- compute the constant term at every cusp;
- prove `Z_(7)` integrality and state the exact `q`-expansion principle;
- carry the away-from-level congruence to the chosen stabilizations, or work in the full Eisenstein ideal;
- show that the ideal generated by all cusp constant terms is a `7`-adic unit ideal.
