# Correction to the t=1 RM/Kummer closure attempt

## Defect in the previous certificate

The script `scripts/p7_t1_rm_rosati_certificate.gp` chose one nonzero left eigenvector for each RM matrix and eigenvalue, formed a scalar projection, and tested whether that scalar lay in the split class or the fundamental-unit line.

This test is not invariant. A left eigenvector over `F_49` is defined only up to multiplication by `F_49^*`; multiplying it rescales the scalar projection. Membership in a fixed `F_7`-line of `F_49` is therefore not preserved by the arbitrary normalization selected by the search order.

Consequently the output

    BAD_COUNT=0

from that script cannot be used to close Lemma D.

## Correct invariant formulation

For a symmetric RM matrix `T` with `T^2-T-I=0` and a root `lambda` of `X^2-X-1`, use the canonical idempotent

    P_lambda = (T-lambda' I)/(lambda-lambda'),

where `lambda'` is the other root. This projector is independent of any eigenvector normalization.

The local period vector must be compared with the projection of the entire globally admissible Selmer subspace, not with a coordinate line after choosing a scalar identification of the one-dimensional eigenspace.

The script

    scripts/p7_t1_rm_projector_audit.py

performs this invariant comparison. It enumerates all eight symmetric matrices in `M_2(F_7)` satisfying the RM polynomial, both eigenvalues, all seven common rational-unit shifts in the two node periods, and the projected `F_7`-line generated by the diagonal fundamental-unit class.

Its exact output is

    ROSATI_COUNT 8
    BAD_COUNT 16
    PROJECTOR_AUDIT_DONE 1

Thus the Rosati polynomial and scalar monodromy form alone do not eliminate the global split/fundamental-unit possibilities.

## Remaining geometric and Diophantine data

A theorem-level local closure now requires at least one of the following refinements.

1. Determine the actual integral RM matrix on the ordered toric node lattice, rather than allowing all eight reductions modulo 7.
2. Determine the actual common rational-unit Kummer shift coming from `A^{-3}` in the primitive `t=1` branch, rather than allowing all seven residue classes.
3. Derive a relation coupling the actual RM matrix and the rational-unit shift from the quotient correspondence or the Raynaud 1-motive.
4. Supply a different reducible-compatible Eisenstein/Selmer obstruction.

The branchwise conductor table, Lemmas A-C, and the finite reduction to the single quadratic survivor `(90,0)` remain valid. Lemma D is not closed by the superseded eigenvector-normalization certificate.
